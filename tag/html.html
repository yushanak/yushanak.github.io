IE下判断IE版本的语句

IE下判断IE版本的语句...[if lte IE 8]……[endif]

<!--[if lte IE 6]>
<![endif]-->
   IE6及其以下版本可见
<!--[if lte IE 7]>
<![endif]-->
   IE7及其以下版本可见
<!--[if IE 6]>
<![endif]-->
   只有IE6版本可见
<!--[if !IE]>
<![endif]-->
   除了IE以外的版本
<!--[if lt IE 8]>
<![endif]-->
   IE8以下的版本可见
<!--[if gte IE 7]>
<![endif]-->
   IE7及大于IE7的版本可见
